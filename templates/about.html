What this website does

This site is intended to be a resource for cooperative gaming. The other major statistics websites for both Dota and League focus almost entirely on "what can I as a player do to beat the other team". I started this project to try and show that there is still market value in getting players to think "cooperative" instead of only "competitive". This site is definitely a work in progress, so until we get a landing page up, here's how to read it:

The first click on the hero / champion grid selects the first hero and displays which items to buy in which order along with their winrate, regardless of their teammates or opponents. Suggested item completion order is top to bottom.

The second click on the grid shows both solo and paired items for both heroes. The selected two can be displayed as either Foes or Friends (on opposite teams and on the same team) using the buttons with the sword icons. You can use how the items change from solo to paired by looking for differences between the columns. You can think of the Teammates / Opponents view as 1v1 and 2v0.

For League, I recommend checking out Annie / Viktor as friends and foes - without being informed about support or carry roles my algorithm picks up on the winning trend of Annie buying support items when paired with Viktor and carry items when fighting him.

For Dota, I recommend Weaver / Ogre Magi - when they're opponents my algorithm suggests Linkens earlier since it deals with Ogre. When they're on the same team the build changes back towards 'normal'

Features we intend to implement -

Pro stats - comparing different levels of skill is easy to do as I gather more match data.

Landing page - this site can be and will get easier to use, we're tackling the web design and UX as we build more features, but there's only two of us.

5v5 calculator - It won't be hard to generalize what I've worked up for the pair calculator into a page showing optimal builds given specified teammates and opponents. I'll implement this when I get a spare day

I'm tweaking the algorithm all the time, so the build order is getting better. If you notice an issue I'd like to hear about it so that I can update around it.

Rationale

ESports are great, and I notice that both Dota and League focus on marketing them to the exclusion of almost everything else. I think that turns away a potential user base of people who want to share the games they love with their friends. This site is being designed to take out the "build" part of Dota and League, so that when new players get dragged in by their friends they have a lower barrier to entry.

Thanks for visiting!
